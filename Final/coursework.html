DEV 109 - Introduction to Web Development

Adam Brehm

Class DEV109 introduces fundamental programming techniques used in current web development. It covers the use of HTML, CSS and Javascript for creating web pages. Before enrolling in this class, I've had some experience developing functional and interactive web pages. Class served as a refresher to me as someone who has some prior experience with these technologies but are looking to update their skills. Overall, the class solidified my knowledge of foundational web development skills while introducing modern practices and tools to enhance web page interactivity and design.
